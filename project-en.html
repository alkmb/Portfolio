Push_swap (Sorting algorithm)

The Push Swap project presents a sorting algorithm to organize numbers in ascending order using two stacks. The challenge lies in the custom implementation of the radix sort algorithm to achieve the desired final order.

Pipex (Implementation of pipes with file descriptors)

The Pipex project introduces a shell implementation of pipes that involves two files and two commands. The main goal is to create a pipe between the commands, where the output of the first command serves as input for the second command.

Fract-ol (Fractal in graphical interface)

The project serves as an introduction to the Xlib library, which provides graphical interfaces in C. The mathematical operations involved in working with real and imaginary numbers to create fractal sets offer an immersive learning experience.

Philosophers (Dining Philosophers Problem)

The Philosophers project is an implementation of the classic Dining Philosophers Problem. It serves as an introduction to multithreading and mutex optimization, providing a practical scenario to understand and solve synchronization problems in concurrent programming.

Minishell (Bash Implementation in C)

Minishell is a bash implementation in C. It offers a deep insight into object-oriented programming and PID management. It provides knowledge about how bash works, including file descriptor redirection, pipe implementation, exit status management, and input parsing for different symbols.

Cub3d (Wolfenstein copy)

Cub3D is a project from École 42 that challenges students to recreate a simplified version of the iconic Wolfenstein 3D game, using the principle of raycasting in C. This project not only tests the students' C programming skills but also introduces advanced concepts of computational graphics and game design. A distinctive feature of Cub3D is the ability to create maps both within the code and externally. Users can specify colors and textures for different elements of the map, as well as adjust the size and shape of the map, offering considerable flexibility in customizing the game environment. This project provides a deep hands-on experience in creating a basic game engine and implementing real-time rendering techniques.

ft_irc (Internet Relay Chat)

ft_irc is a client-server communication project where a Singleton is implemented to access objects in real-time and update the server according to the commands sent by the client. Its implementation includes the creation of channels and the option to send messages between clients and within channels. It also includes permission management to invite and expel clients, and the implementation of the MODE command to configure the channel's parameters and rules.
