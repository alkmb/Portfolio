1 - Libft (Library of functions in C)

Libft is a library with C functions (some are modified implementations of existing functions) that offers essential functionalities, such as memory manipulation, type conversion, string manipulation, list operations, and more.

2 - ft_printf (printf copy)

The ft_printf project is a simplified replica of the original printf function in C. The main focus is to create a compact but robust utility that handles a limited set of format specifiers: %c, %s, %d, %i, %u, %p, %x, and %X.

3 - Get_next_line (Memory managment)

This project is about reading and storing lines from a file into a buffer in C. The main goal is to acquire competence in memory allocation and release, avoiding memory leaks while providing a flexible and reliable solution for file input handling.

4 - Push_swap (Sorting algorithm)

The Push Swap project presents a sorting algorithm to organize numbers in ascending order using two stacks. The challenge lies in the custom implementation of the radix sort algorithm to achieve the desired final order.

5 - Pipex (Implementation of pipes with file descriptors)

The Pipex project introduces a shell implementation of pipes that involves two files and two commands. The main goal is to create a pipe between the commands, where the output of the first command serves as input for the second command.

6 - Fract-ol (Fractal in graphical interface)

The project serves as an introduction to the Xlib library, which provides graphical interfaces in C. The mathematical operations involved in working with real and imaginary numbers to create fractal sets offer an immersive learning experience.

7 - Philosophers (Dining Philosophers Problem)

The Philosophers project is an implementation of the classic Dining Philosophers Problem. It serves as an introduction to multithreading and mutex optimization, providing a practical scenario to understand and solve synchronization problems in concurrent programming.

8 - Minishell (Bash Implementation in C)

Minishell is a bash implementation in C. It offers a deep insight into object-oriented programming and PID management. It provides knowledge about how bash works, including file descriptor redirection, pipe implementation, exit status management, and input parsing for different symbols.
