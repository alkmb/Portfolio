1 - Libft (Libreria de funciones de C)

Libft es una biblioteca con funciones de C (algunas son implementaciones modificadas de funciones existentes) que ofrece funcionalidades esenciales, como manipulación de memoria, conversión de tipos, manipulación de cadenas, operaciones de listas y más.

2 - ft_printf (Replica de printf)

El proyecto ft_printf es una réplica simplificada de la función printf original en C. El enfoque principal es crear una utilidad compacta pero robusta que maneje un conjunto limitado de especificadores de formato: %c, %s, %d, %i, %u, %p, %x, %X y %%. Este proyecto sirve como una introducción al análisis básico, utiliza la macro va_arg para argumentos variables y pone énfasis en el manejo de errores para mejorar la comprensión y maestría de los conceptos de programación en C.

3 - Get_next_line (Manejo de memoria)

Este proyecto se centra en la creación de una utilidad simple y eficiente llamada Get Next Line (GNL), diseñada para leer y almacenar líneas de un archivo en un búfer en C. El objetivo principal es adquirir competencia en la asignación y liberación de memoria, evitando fugas de memoria mientras se proporciona una solución flexible y confiable para el manejo de la entrada de archivos.

4 - Push_swap (Algoritmo de ordenacion)

El proyecto Push Swap presenta un algoritmo de ordenación para organizar números en orden ascendente utilizando dos pilas. El método emplea un proceso de dos pasos: primero, se aplica una versión simplificada del algoritmo de ordenación de burbuja para ordenar inicialmente los números, seguido de una implementación personalizada del algoritmo de ordenación de radix para lograr el orden final deseado. El proceso de ordenación implica rotaciones y empujes, optimizando la estrategia de ordenación para obtener eficiencia.

5 - Pipex (Implementacion the pipes con file descriptors)

El proyecto Pipex introduce una implementación de shell de tuberías que implica dos archivos y dos comandos. El objetivo principal es crear una tubería entre los comandos, donde la salida del primer comando sirve como entrada para el segundo comando. El resultado final se almacena en el segundo archivo. Este proyecto está diseñado para familiarizarte con la función execve para ejecutar los ejecutables correctos, navegar por las variables de entorno y manejar varios comandos en la parte de bonificación.

6 - Fract-ol (Fractal en interfaz grafica)

El proyecto Fract-ol es un programa visualmente cautivador que muestra fractales, centrándose especialmente en los conjuntos de Mandelbrot y Julia. El proyecto sirve como una introducción a la biblioteca Xlib, que proporciona interfaces gráficas en C. Las operaciones matemáticas involucradas en el trabajo con números reales e imaginarios para crear los conjuntos fractales ofrecen una experiencia de aprendizaje inmersiva. Las características adicionales incluyen opciones de cambio de color, zoom siguiendo el cursor, movimiento con flechas y la adición de un tercer fractal, el Tricorn.

7 - Philosophers (Dining Philosophers Problem)

The Philosophers project is an implementation of the classic Dining Philosophers problem. It serves as an introduction to multithreading and mutex optimization, providing a practical scenario to understand and solve synchronization issues in concurrent programming. It's also a good introduction to object-oriented programming concepts.

8 - Minishell (Bash Implementation in C)

Minishell is an implementation of bash written in C. It provides a deep dive into object-oriented programming and PID management. It's a great introduction to how bash works behind the scenes, managing file descriptor redirection, pipe implementation, exit status management, rewriting built-in commands, and parsing input for different symbols.
