Push_swap (Algoritmo de ordenacion)

El proyecto Push Swap presenta un algoritmo de ordenación para organizar números en orden ascendente utilizando dos pilas. El desafío radica en la implementación personalizada del algoritmo de ordenación de radix para lograr el orden final deseado.

Pipex (Implementation of pipes with file descriptors)

El proyecto Pipex introduce una implementación de shell de tuberías que involucra dos archivos y dos comandos. El objetivo principal es crear una tubería entre los comandos, donde la salida del primer comando sirve como entrada para el segundo comando.

Fract-ol (Fractal en interfaz grafica)

El proyecto sirve como una introducción a la biblioteca Xlib, que proporciona interfaces gráficas en C. Las operaciones matemáticas involucradas en el trabajo con números reales e imaginarios para crear los conjuntos fractales ofrecen una experiencia de aprendizaje inmersiva.

Philosophers (Dining Philosophers Problem)

El proyecto de Philosophers es una implementación del clásico problema de los Filósofos Comensales. Sirve como una introducción al multithreading y la optimización de mutex, proporcionando un escenario práctico para entender y resolver problemas de sincronización en la programación concurrente.

Minishell (Bash Implementation in C)

Minishell es una implementación de bash en C. Ofrece una visión profunda de la programación orientada a objetos y la gestión de PID. Proporciona conocimientos sobre cómo funciona bash, incluyendo la redirección de descriptores de archivos, implementación de tuberías, gestión de estado de salida, y análisis de entrada para diferentes símbolos.

Cub3d (Wolfenstein copy)

Cub3D es un proyecto de la École 42 que desafía a los estudiantes a recrear una versión simplificada del icónico juego Wolfenstein 3D, utilizando el principio de raycasting en C. Este proyecto no solo pone a prueba las habilidades de programación en C de los estudiantes, sino que también introduce conceptos avanzados de gráficos computacionales y diseño de juegos. Una característica distintiva de Cub3D es la capacidad de crear mapas tanto dentro del código como externamente. Los usuarios pueden especificar colores y texturas para diferentes elementos del mapa, así como ajustar el tamaño y la forma del mapa, ofreciendo una flexibilidad considerable en la personalización del entorno de juego. Este proyecto proporciona una experiencia práctica profunda en la creación de un motor de juego básico y la implementación de técnicas de renderizado en tiempo real.

ft_irc (Internet Relay Chat)

ft_irc es un proyecto de comunicación cliente-servidor donde se implementa un Singleton para poder acceder a los objetos en tiempo real y actualizar el servidor conforme a los comandos enviados por el cliente. Su implementación incluye la creación de canales y la opción de enviar mensajes entre clientes y dentro de canales. También incluye manejo de permisos para poder invitar y expulsar clientes, y la implementación del comando MODE para poder configurar los parámetros y reglas del canal.
