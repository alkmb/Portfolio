Proyectos

1 - Libft (Libreria de funciones de C)

Libft es una biblioteca con funciones de C (algunas son implementaciones modificadas de funciones existentes) que ofrece funcionalidades esenciales, como manipulación de memoria, conversión de tipos, manipulación de cadenas, operaciones de listas y más.

2 - ft_printf (Replica de printf)

El proyecto ft_printf es una réplica simplificada de la función printf original en C. El enfoque principal es crear una utilidad compacta pero robusta que maneje un conjunto limitado de especificadores de formato: %c, %s, %d, %i, %u, %p, %x y %X

3 - Get_next_line (Manejo de memoria)

Este proyecto se leer y almacenar líneas de un archivo en un búfer en C. El objetivo principal es adquirir competencia en la asignación y liberación de memoria, evitando fugas de memoria mientras se proporciona una solución flexible y confiable para el manejo de la entrada de archivos.

4 - Push_swap (Algoritmo de ordenacion)

El proyecto Push Swap presenta un algoritmo de ordenación para organizar números en orden ascendente utilizando dos pilas. El desafío radica en la implementación personalizada del algoritmo de ordenación de radix para lograr el orden final deseado.

5 - Pipex (Implementacion the pipes con file descriptors)

El proyecto Pipex introduce una implementación de shell de tuberías que implica dos archivos y dos comandos. El objetivo principal es crear una tubería entre los comandos, donde la salida del primer comando sirve como entrada para el segundo comando.

6 - Fract-ol (Fractal en interfaz grafica)

El proyecto sirve como una introducción a la biblioteca Xlib, que proporciona interfaces gráficas en C. Las operaciones matemáticas involucradas en el trabajo con números reales e imaginarios para crear los conjuntos fractales ofrecen una experiencia de aprendizaje inmersiva.

7 - Philosophers (Dining Philosophers Problem)

El proyecto de Philosophers es una implementación del clásico problema de los Filósofos Comensales. Sirve como una introducción al multithreading y la optimización de mutex, proporcionando un escenario práctico para entender y resolver problemas de sincronización en la programación concurrente.

8 - Minishell (Bash Implementation in C)

Minishell es una implementación de bash en C. Ofrece una visión profunda de la programación orientada a objetos y la gestión de PID. Proporciona conocimientos sobre cómo funciona bash, incluyendo la redirección de descriptores de archivos, implementación de tuberías, gestión de estado de salida, y análisis de entrada para diferentes símbolos.
